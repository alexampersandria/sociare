# Sociare

A WeShare replacement API.

---

It is recommend to use [rust-analyzer](https://rust-analyzer.github.io/manual.html) for development. And change the `rust-analyzer.checkOnSave.command` to `clippy`.

## Run

### Cargo

```bash
cargo run
```

## Test

### Cargo

```bash
cargo test
```

## Lint

### Cargo

```bash
cargo fmt
cargo clippy
```

## Build

### Cargo

```bash
cargo build
```
